## Quick Reference: Emission Model Decision Tree

```
AGN X-ray Spectrum Observed
├─ Hard X-ray Photon Index Γ?
│  ├─ Γ < 1.7 (hard spectrum)
│  │  └─ → Hot corona dominates, low accretion rate, possibly high spin
│  ├─ Γ ~ 1.9 (typical)
│  │  └─ → Mixed disk-corona, sub-Eddington accretion
│  └─ Γ > 2.2 (soft spectrum)
│     └─ → Disk-dominated, high accretion rate, possibly low spin
│
├─ Soft Excess Present?
│  ├─ Yes (Γ_soft > 2.5, flux ~10–30% hard continuum)
│  │  ├─ UV-optical correlation? → Warm Corona Model
│  │  ├─ High-density disk signature? → Ionized Reflection Model
│  │  └─ Both? → Hybrid Model
│  └─ No → Pure hot corona, low Eddington ratio
│
├─ Iron K-line Features?
│  ├─ Broad line (FWHM > 1 keV) → Inner disk reflection, measure spin
│  ├─ Narrow line → Distant torus, high inclination obscuration
│  └─ No detection → Low reflection component, edge-on viewing or LLAGN
│
├─ Variability Timescale?
│  ├─ Minutes → Ultra-compact corona (R < 10^3 km), magnetic reconnection
│  ├─ Hours → Compact corona (R ~ 10^4 km), flare model
│  └─ Days+ → Propagating disk instabilities, outer regions coupled to X-rays
│
└─ Inference → Accretion state, black hole spin, coronal geometry, heating mechanism
```

## Notation and Conventions

- **M_⊙**: Solar mass (1.989 × 10^33 g)
- **R_g**: Gravitational radius, R_g = GM_BH / c^2 ~ 1.48 × 10^5 (M_BH / 10^8 M_⊙) cm
- **Schwarzschild radius**: R_s = 2 R_g
- **Eddington luminosity**: L_Edd = 4πGM c / κ_e ~ 1.26 × 10^38 (M / M_⊙) erg s^−1
- **Eddington ratio**: λ_Edd = L_bol / L_Edd (dimensionless)
- **Compton parameter**: y = 4 k_B T_e τ / (m_e c^2)
- **Thomson cross-section**: σ_T ~ 6.65 × 10^−25 cm^2
- **Doppler factor**: δ = 1 / [Γ(1 − β cos θ)]
- **Bulk Lorentz factor**: Γ_bulk = 1 / √(1 − β^2)
- **Photon index**: dN/dE ∝ E^{−Γ}; Γ = 2 is canonical
