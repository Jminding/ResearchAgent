KEY FINDINGS DOCUMENTED (12 Total)
================================================================================

F1: Extreme Class Imbalance is Fundamental
   - IEEE-CIS: 3.49% fraud (27.5:1)
   - Kaggle: 0.17% fraud (578:1)
   - Production: 0.0005-0.05% (2000-200000:1)

F2: High Accuracy Does Not Guarantee Real-World Effectiveness
   - Test AUC: 99-100%
   - Real-world FPR: 90-98%

F3: Isolation Forest Baseline Highly Competitive
   - Accuracy: 0.85-0.95
   - Time Complexity: O(n log n)
   - Unsupervised (no labels required)

F4: Graph Structure Provides 2-10% Performance Gain
   - GNN improvement over feature-only: 2-10% F1-score
   - Network-aware detection essential

F5: Temporal Dynamics Critical for Money Laundering
   - Multi-phase structure: placement → layering → integration
   - TGN significantly outperforms static GNNs

F6: Synthetic AML Datasets Enable Complete Ground Truth
   - eMoney and SAML-D with complete labels
   - Real data has undetected laundering (label noise)

F7: False Positive Rates Dominate Operational Costs
   - Traditional: 90-98% FPR
   - Advanced graph-based: 10-20% FPR
   - Annual AML compliance cost: $274 billion

F8: Banking Networks are Extremely Sparse and Large
   - 1.6M nodes, 3.8M edges
   - Density: 0.00014%
   - GPU scalability challenges

F9: Downsampling Most Effective for Large Data
   - +0.5% AUC improvement on IEEE-CIS
   - Preferred over SMOTE for >100K samples

F10: Graph Autoencoders Effective for Unsupervised Detection
   - LG-VGAE: +3.7% precision, +7.0% recall, +5.7% F1
   - No labeled data required

F11: Ensemble Methods Achieve Highest Accuracy
   - Stacking (XGBoost+LightGBM+CatBoost): 99.94% accuracy, 100% AUC
   - State-of-the-art for supervised settings

F12: 28+ Money Laundering Typologies Identified
   - Placement, layering, integration phases
   - Multi-class detection framework needed

CLASS IMBALANCE MITIGATION STRATEGIES (4 Total)
================================================================================

1. Downsampling
   - Effectiveness: +0.5% AUC
   - Best For: >100K samples
   - Recommendation: Primary choice

2. SMOTE
   - Effectiveness: +5% F1-score
   - Best For: 5-50:1 imbalance
   - Risk: Synthetic correlations

3. Cost-Weighted Learning
   - Effectiveness: +3-5% F1-score
   - Methods: XGBoost scale_pos_weight, RF class_weight
   - Recommendation: Use with boosting

4. Stratified K-Fold Cross-Validation
   - Effectiveness: Essential for reliable estimates
   - Standard: 5-fold
   - Always use with temporal holdout
